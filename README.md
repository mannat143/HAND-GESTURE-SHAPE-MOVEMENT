# HAND-GESTURE-SHAPE-MOVEMENT

This project is a web-based interactive visual application that uses 3D graphics and real-time hand tracking to control particle animations. It combines Three.js for rendering 3D particles and MediaPipe Hands for detecting and interpreting hand gestures through a webcam.

The system creates a dynamic environment where users can interact with thousands of particles using simple hand movements, making the experience immersive and visually engaging.

Hand Gesture Control

The project uses real-time hand tracking to control particle behavior:

Pinch / Spread fingers → Scale particles (zoom in/out)
Move hand → Change particle colors dynamically
Open / Close hand → Morph shapes (sphere ↔ heart)
